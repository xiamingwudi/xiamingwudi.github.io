函数重载分析

发表于 2017-12-06 | 分类于 C++

重载的概念

重载（overload）：同一个标示符在不同上下文具有不同的含义。

比如：“洗”和不同的词汇搭配具有不同的含义，如洗衣服、洗脸、洗脑等等。

重载这个概念在自然语言中随处可见，那么程序设计中是否也包含重载呢？

C++的函数重载

函数重载

函数重载：用一个函数名定义不同的函数，当函数名和不同的参数搭配时函数的意义不同。

函数重载至少满足下面的一个条件：

  1. 参数个数不同
  2. 参数类型不同
  3. 参数顺序不同

编译调用重载函数的准则

  1. 将所以同名函数作为候选者
  2. 尝试寻找可行的候选函数
  • 精确匹配实参
  • 通过默认参数匹配实参
  • 通过默认类型转换匹配实参
  1. 匹配失败
  • 最终寻找到的候选函数不唯一，则出现二义性，编译失败
  • 无法匹配所有的后选者，函数未定义，编译失败

函数重载的注意事项

  1. 函数重载的本质是相互独立的不同函数
  2. 重载函数的函数类型是不同的
  3. 函数返回值不能作为函数重载的依据
  4. 函数重载是由函数名和参数列表决定
